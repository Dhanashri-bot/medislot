# 🏥 MediSlot – Clinic Management System

MediSlot is a full-stack web application for managing clinic-related activities. It allows users to view doctors and access basic healthcare features through a simple and responsive interface.

---

## 🚀 Features
- View list of doctors
- User authentication (Login system)
- REST API integration
- Responsive UI
- Admin access support

---

## 🛠️ Tech Stack

**Frontend:**
- React
- Vite

**Backend:**
- Django
- Django REST Framework

---

## 🌐 Deployment

- Frontend deployed on Vercel:  
https://medislot-ashen.vercel.app/

- Backend was developed and tested locally (not deployed)
